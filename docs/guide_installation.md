# Guide d'installation

## Prérequis
- Python 3.8 ou supérieur

## Installation

1. Cloner le dépôt :
   git clone https://github.com/galantadame-del/SAE203_Groupe2.git

2. Installer les dépendances :
   pip install -r requirements.txt

3. Configurer les cibles dans data/cibles.json

4. Lancer le programme :
   python main.py
